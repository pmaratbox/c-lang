# 0007 — Collections

Build an array of the integers `1, 2, 3, 4, 5`, then print its count and its
first and last elements. A raw array doesn't carry its length, so it is
computed as `sizeof(nums) / sizeof(nums[0])`. Indexing is pointer arithmetic
with no bounds checking — reading past the end is undefined behavior.

## Run

    make && ./collections
