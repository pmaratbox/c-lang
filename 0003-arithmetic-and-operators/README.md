# 0003 — Arithmetic & operators

Compute and print the five elementary arithmetic operations on two integers
(`a = 17`, `b = 5`). C's `/` between two `int`s truncates toward zero (since
C99; earlier C left rounding implementation-defined). Use `(double)a / b` to
get the fractional quotient. `%%` in the format string prints a literal `%`.

## Run

    make && ./arithmetic
